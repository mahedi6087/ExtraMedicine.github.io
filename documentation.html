Project Summary

This project is a web app that helps people find Halal restaurants and shops in South Korea. It uses Google Maps to show places on a map, lets users search by location or use GPS, and shows names and distances in a sidebar.

üß†

Problem It Solves

Muslim tourists and residents in Korea often struggle to find Halal food because:

  • They don't know the area well
  • Language is a barrier
  • Most apps don't have a Halal filter
  • Regular map apps aren't made for Halal needs
Solution: This project solves that problem by giving a simple, Halal-focused map tool.

How It Works

1
Location Input: The user can enter a city/district (like "Seoul" or "Busan") or click "Use My Location".
2
Coordinate Processing: The app either geocodes the typed area or uses GPS to get coordinates.
3
API Query: The system sends a query to Google Places API like "halal restaurant near [location]".
4
Results Display: The map displays all results with markers.
5
Interactive Features: The sidebar shows a scrollable list of places with name + distance. On mobile, users can swipe place cards and tap to highlight the map.
